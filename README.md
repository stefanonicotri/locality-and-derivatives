# Mathematical Physics Notes: Locality and Derivative Operators

A standalone, beautifully formatted, and interactive HTML document exploring the mathematical boundaries between local and nonlocal operators in theoretical physics. 

## 📖 Overview

In field theory, quantum mechanics, and probability, the distinction between **local** and **nonlocal** operators dictates how information, forces, and particles spread through space. 

This document is entirely self-contained in a single `index.html` file.
